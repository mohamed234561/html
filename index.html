Large files are stored in a large database and contain all recorded data that can be used at a later time. This base is called a Database. Because databases are very important and influential in all major areas and activities. Therefore, certain systems are required to organize and manage the stored data. This is what is called Database Management Systems, abbreviated as DBMS. Database management systems are defined as the programs that help create databases, deal with them, and run the data stored in them. For example, after adding new customers to the phone book, you need to arrange the names again alphabetically or arrange their addresses. In other words, it allows the user to add new data, update the data, and print reports in the format they want, such as lists, tables, forms, invoices...etc. The database may be very large and contain thousands of billions of words, and it is larger than the existing memory. As a result, the DBMS had to process and manage data in the secondary memory. There are many programs designed for this purpose, including those that run on mainframes or on personal computers. PCs.

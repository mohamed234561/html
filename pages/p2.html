Database Management Systems

- Hierarchical DBMS Network DBMS management systems Relational DBMS Hierarchical DBMS: Hierarchical databases or hierarchical systems DBMS organize data in a hierarchical form or in the form of an inverted tree, that is, its root is at the top and branches come out of it. This structure is like the family tree, it has one grandfather and the grandfather has several sons, and the sons are the fathers of the grandchildren, and it is impossible for a grandson to have more than one father. This is an illustration to show you the hierarchical systems and their branches Hierarchical files are files that have the same tree structure and have the same relationships between records, an example of some types of records that can exist in a hierarchical configuration, there are multiple sales records for each seller where there is one statistics record for each ongoing process, and there are also many records for customers for each seller as each seller It has specific customers, and each customer can have several accounts receivable records, one record for each unpaid purchase. It is important to understand that it is not necessary for all the files in the database to communicate with each other. All that is required is that the files that are used as a group communicate with each other in the applications. And past sales records have such a logical relationship called category. The Set class is a set of records connected to each other logically. Thus, a hierarchical database is a collection of files and file categories that are logically connected to each other. The information management system (IMS), prepared by IBM, uses hierarchical configuration, which is one of the largest and most complex dbms database management systems currently in existence. For this reason, it requires a high level of expertise to be able to build it, and in any case, it is strong and has proven great efficiency in dealing with very large databases. It also provides good retrieval and security procedures, in addition to the possibility of using it in the active communication system through the communication network.
